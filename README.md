How to use: 

-- SFIND 
Find strings inside filtered files. 
E.g:
Find a string in java files ... 
$ sfind java getAvalue

-- FFIND 
Search for files
E.g: 
Looking for readme files
$ ffind Readme*


-- ANDROIDSTRING
Look strings thorugh all android resource files
E.g: 
Find log_title string
$ androidstring log_title
